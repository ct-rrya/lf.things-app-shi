# Flowchart Quick Start Guide

## Need to Print Flowcharts? 🖨️

### 3 Simple Steps:

1. **Choose a flowchart** from `docs/graphviz/`:
   - `print-00-overview.dot` - Best for presentations (1 page)
   - `print-00-complete-system.dot` - Full system overview
   - `print-01-authentication.dot` - Sign in/up flow
   - `print-02-item-registration.dot` - Register items
   - `print-03-qr-scanning.dot` - QR scanning
   - `print-04-report-found.dot` - Report found items
   - `print-05-ai-matching.dot` - AI matching
   - `print-10-admin-dashboard.dot` - Admin features
   - `print-14-home-dashboard.dot` - Home screen

2. **Generate PDF**:
   - Go to https://dreampuf.github.io/GraphvizOnline/
   - Copy & paste the file content
   - Click "Export" → "PDF"

3. **Print**:
   - Use landscape orientation
   - 300 DPI quality
   - Color or grayscale both work

## What's Different?
