## 🖨️ Need to Print Flowcharts?

**Quick Start:**
1. Go to `docs/graphviz/`
2. Use `print-00-overview.dot` (single-page overview)
3. Open https://dreampuf.github.io/GraphvizOnline/
4. Copy & paste file content
5. Export as PDF and print

**Detailed Guide:** See [PRINTING_FLOWCHARTS_GUIDE.md](PRINTING_FLOWCHARTS_GUIDE.md)
